Table tag

웹 문서에서 Table 형태로 자료를 정리하기 위해 사용되는 태그

행과 열로 구성되어 있고, 만나는 지점을 셀이라 한다.

table 태그 : 테이블을 지칭하는 태그로 가장 밖에 선언된다.

tr 태그 : 한개의 행(row)을 나태내는 태그

td 태그 : 한개의 열(col)을 나태내는 태그

테이블 예시

홍길동 23살
최길동 32살

border 속성 : 테이블 테두리 굵기를 조정해주는 속성

홍길동 23살 010-1234-1234
최길동 32살 010-9999-3648

caption 태그 : 테이블의 제목을 지정하는 태그

회원 정보
홍길동 23살 010-1234-1234
최길동 32살 010-9999-3648

th 태그 : 테이블의 제목 셀을 지정하는 태그

회원 정보
이름 나이 전화번호
최길동 32살 010-9999-3648

colspan 태그 : 열 병합하는 태그

안녕 하이
안녕하세요

하이2
하이3
