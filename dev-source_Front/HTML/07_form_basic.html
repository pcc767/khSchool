form 관련 태그

- 사용자로부터 text 및 각종 입력을 전달받아 page가 지정한 서버로 전달하는 태그

- 주요한 form : text, pw, textarea, button, summit, readio, select 등

from 기초 예제

- action : 전달할 서버의 url

- method : 전달할 방식을 선택, get은 url을 통한 노출, post는 url과 별개로 데이터를 전달.
