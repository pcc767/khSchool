폼 입력 양식 요소

input tag

- 사용자의 입력을 위한 다양한 Type을 갖는 입력 필드를 생성하는 태그


기본 Type에 대한 속성

text Type 속성

- 한줄짜리 입력창이 생기는 속성, 가장 기본적인 필드

- name : 서버에서 클라이언트 요청 파라메터(데이터)를 인지할 식별자(이름)

- id : 브라우저에서 페이지의 태그를 인지할 식별자(id), 주로 js나 css에서 식별 용도로 활용



submit / reset Type

submit : 입력한 데이터를 서버 url로 제출하는 기능, 버튼 외에도 js로도 같은 기능을 할 수 있음

reset : 입력한 데이터를 초기화 하는 기능, form 단위로 리셋됨

hidden Type attribute

- 사용자에게 보여지지 않고 클라이언트 전달됐다가 다시 서버로 입력될수 있는 파라메터

- 주로 개발 편의성으로 사용되나 가끔 보안의 헛점이 될 수 있음으로 주의 요구

checkbox Type attribute

- name을 동일한 그룹으로 설정하고 다중 선택이 가능한 속성.

- 향후 서버 사이드에서 가장 다루기 귀찮은 설정 --> 다르게 설정하는 것을 추천.

자바 오라클 html

radio Type attribute

- name을 동일한 그룹으로 설정하고, 단일 선택만 가능한 속성.

자바 오라클 html

file Type attribute

- 파일을 전송하기 위한 속성, 단일이나 다중 선택이 가능하다. --> 대부분이 단일만 지원

image Type attribute

- 사실상 이미지 버튼용도로 활용되는 태그

Button Type attribute

- button으로 활용, submit이나 onclick 이벤트 처리하는 용도로 활용


type HTML5 양식 태그

- HTML5로 와서 적용된 태그로 IE나 올드한 version에서는 적용되지 않음

- HTML5의 표준을 잘 지키는 크롬계열의 브라우저(크롬, 웨일, 사파리, 엣지)에서는 사용 가능

날짜 관련 type 속성

date Type 속성

- 년, 월, 일 선택 가능한 양식 제공


date Type-local Type attribute

- 년, 월, 일, 시간 선택 가능한 양식 제공


month type attribute

- 년, 월 선택 가능한 양식제공



week type attribute

- 주 선택 가능한 양식제공



time type attribute

- 시간만 선택 가능한 양식 제공

기타 Type attribute

color type 속성

number type 속성

숫자 입력 양식

range type 속성

슬라이드 막대로 숫자 지정 양식

search type 속성

검색어 입력 양식(입력 후 x버튼 생김)

email type 속성

데이터 전송 시 @를 확인하는 입력 양식

url type 속성

데이터 전송 시 주소인지(http://)를 확인하는 입력 양식

그외 input 태그의 속성

placeholder 속성

- 사용자의 입력창에 hint(뒤에 회색 글씨)로 표시하는 기능

readonly 속성

- 읽기전용으로 사용자가 값을 변경할 수 없음

required 속성

- submit시 필수 입력값으로 지정하여 브라우저에서 체크하는 항목
