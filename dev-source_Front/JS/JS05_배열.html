배열의 선언

배열의 선언과 초기화 방법


배열 주요 메소드(기능)

push() : 배열의 마지막 index에 값 넣기

pop() : 배열의 마지막 index에 값 빼기

unshift() : 배열의 처음 index에 값 넣기

shift() : 배열의 처음 index에 값 빼기


indexOf() : 배열에서 요소가 위치한 인덱스를 찾아 리턴, 값이 없으면 -1(음수)


concat() : 두개 또는 세개의 배열을 결합


join() : 배열을 결합하여 문자열로 반환


reverse() : 배열의 순서를 뒤집는 메소드


sort() : 배열의 값을 내림차순, 오름차순으로 정렬하는 함수★★★★★

- 특징 : 알아서 내림차순으로 정렬하지만, 사용자가 우선순위 지정도 가능함
※ 주의점 : 기본 sort는 문자열 기준으로 정렬됨!! -> 숫자 제대로 정렬 안됨!!!

slice() : 배열의 요소를 잘라내는 메소드, 원본배열 변화 없음

splice() : 배열의 요소를 완벽하게 제거하는 메소드=(delete, remove), 원본 배열 변화 존재!


toString() : 배열을 문자열로 반환


find() : 조건을 찾아서 반환하는 함수
