데이터 입출력

- 자바스크립트를 통해 html이나 콘솔을 통해 사용자 데이터를 입출력하는 방식

데이터 출력법

1. document.write() 사용

- 브라우저상으로 문구를 출력하는 기본적인 방법

2. window.alert()

- 별도의 브라우저 메세지 창으로 출력하는 방법.(window 생략 가능)
- ※ 주의 : 사용성이 좋지 않아 최근 스타일에는 맞지 않는 기법, 학습용/디버깅 권장!!

3. innerHTML★★★★

- HTML 태그 객체를 지정하여 출력하는 방법
- 주로 id로 찾아서 객체를 조작하는 것이 일반적
기본 문장

4. console.log

- 개발자 도구에서 console로 출력하는 방법이다.
- 일반 사용자에게 보여지지 않으며, 개발이 끝나면 지우는 것이 바람직하다.!

데이터 입력법

1. HTML input태그를 통해 읽어오는 방법★★★★

- HTML 태그 중 input이 가능한 태그에서 사용자 입력 값을 읽어오는 방법
- 보통 submin(서버전달)을 하기 전에 js 이벤트를 받아 처리하는 것이 일반적

2. window.confirm()

- 사용자에게 메세지 창으로 Y/N으로 확인 받는 방법, 요새는 안쓴다.!!!

3. window.prompt()

- 사용자에게 텍스트 값을 입력받는 방법, 요새는 안쓴다!!2
