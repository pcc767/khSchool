Window 객체

- Window 객체는 자바스크립트의 최상위 객체로써 하위 DOM, BOM을 가지고 있음

window.open()

- 특정 URL을 오픈하는 기능


Window 객체의 Timer 메소드

setTimeout()

- 일정시간 이후에 예약된 함수를 한번 실행시키는 구조

- window.setTimeout("실행함수", 시간); 시간=ms 단위

- 시간이벤트에 대한 callback 함수 구조를 가진다.

setInterval()

- 일정시간마다 지정한 함수를 반복하는 방법

- window.setInterval("실행 함수", 시간); 시간은 ms단위로 설정


clearInterval()

- Interval 종료시키는 함수
