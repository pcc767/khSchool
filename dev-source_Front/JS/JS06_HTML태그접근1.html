1. getElementById() ★★★★★

- ID인자를 통해 태그 객체를 가져오는 방법으로 js에서 가장 보편적으로 활용됨!!


2. getElementsByName() : 이름으로 가져오기

- name을 통해서 태그 객체'들'을 가져오는 방법 (Element's'가 붙으면 배열로 가져온다!)


3. getElementsByTagName

- 태그 이름 값을 통해 객체들을 가져오는 방법


4. getElementsByClassName

- Class의 이름을 가진 객체들을 가져오는 방법

5. 요소값 변경하기

- 가져온 객체를 통해 내부 요소를 변경할 수 있음
