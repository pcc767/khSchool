자바스크립트의 개요

- 웹브라우저에서 HTML문서에 대한 동적로직 처리가 가능한 인터프리터 방식의 언어
- 객체지향 문법을 지원하고, 실질적으로 절차지향으로도 프로그래밍이 가능한 언어
- 실제 동작하는 방식은 브라우저에 존재하는 웹 엔진(크론 엔진)을 통해 파싱 및 실행 가능

자바스크립트의 선언 방법

- html 파일 내에 script 태그를 활용하여 JavaScript 영역을 생성함.
- type="text/javascript"를 포함 해야 하지만 default가 js라 생략 가능

자바스크립트 코드의 위치

- 페이지 내에서 어디서든 가능하지만, head, body 태그 내부에 선언하는 것이 일반적

자바스크립트 작석 유형

1. Internal(내부) 방식

- 일반적으로 body나 head에 선언하는 방식을 말함.

2. Inline 방식

- html 태그 이벤트 핸들어 속성을 이용하여 javascript 코드를 실행하는 방법

예시1 : 인라인 방식으로 html 태그 내부에서 선언하는 방식 -> 아주 짧은 코드만 추천.

예시2 : 미리 선언한 함수를 통해 호출하는 방법★★★★★★★★


3. External(외부) 에서 호출되는 방법

- 자바스크립트 코드를 외부에서 작성하여 로드하는 방식
- 파일 확장자는 .js로 끝남 단, .js가 아니어도 로드 가능
