함수 (function)

- 함수란 입력(매개변수)과 출력(return or 직접 출력)을 통해 특정 기능을 구현하기 위한 문법

1. 함수의 선언법

- function functionName([parameters]) {처리 코드; [return]}


2. 익명함수 선언법(=일반 함수로 해도 동일하게 기능적용 가능!)

- var functionName1 = function [functionName2] ([parameters]) {처리 코드; [return]}

- 활용 팁 : 이벤트 핸들러, 함수가 인자로 들어갈때, 임시적인 함수 선언시 활용됨


3. 스스로 실행하는 익명함수 = 사용하지 않는다!

"(function (parameters) { 처리 코드 })();"
