여러값을 표현하는 태그

select tag

- 드롭다운으로 목록의 선택을 유도하는 태그

option 태그 : select에 종속되어 옵션 값을 지정해주는 태그

optgroup 태그 : option을 그룹핑한 태그 단, 선택되지 않음.

1. 기본적인 select 활용

2. size를 통하여 화면에 보여줄 갯수를 지정하는 방법

3. multiple : 여러 항목을 ctrl을 누른체 선택할 수 있는 방법

4. optgroup을 활용하여 그룹핑을 하는 방법

datalist 태그

- input 태그의 리스트 형식을 제공하여 옵션을 자동을 입력하는 기능 (절대적이지 않음!!)

textarea 태그

- 여러줄을 입력받을때 사용하는 태그, cols:줄의 길이, rows : 행의 길이를 표기 해야함

- 태그 시작과 끝 사이 공간이 실제 컨텐츠가 됨으로 공백 유지 필수!
