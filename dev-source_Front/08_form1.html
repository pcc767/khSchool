form tag

- 사용자가 입력한 값에 대한 처리 방식과 실제 처리할 서버의 url을 명시하는 태그

- action : 실제 서버가 처리해야할 url 혹은 주소정보를 기입

- method : get/post를 통해 서버에 전달할 방식으로 기입(default : get)

- submit : form의 입력된 데이터를 서버로 전달하는 트리거(버튼) - input의 속성





fieldset, legend tag(나중가면 안쓰는 태그)

fieldset : form 요소의 그룹을 묶어주는 태그

legend : form 요소 그룹의 명칭을 붙여주는 태그
