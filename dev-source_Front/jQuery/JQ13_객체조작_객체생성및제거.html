append : 객체 삽입해보기

- $(A).append(B) : B를 A의 요소 내 뒷부분에 추가 - 자식부모

- $(A).prepend(B) : B를 A의 요소 내 앞부분에 추가 - 자식부모

- $(A).After(B) : B를 A의 요소 외 뒤에 추가하는 법 - 형제

- $(A).Before(B) : B를 A의 요소 외 앞에 추가하는 법 - 형제

appendTo() : 객체 삽입해보기

- $(A).appendTo(B) : A를 B의 요소 내 뒷부분에 추가 - 자식부모

- $(A).prependTo(B) : A를 B의 요소 내 앞부분에 추가 - 자식부모

- $(A).insertAfter(B) : A를 B의 요소 외 뒤에 추가하는 법 - 친구

- $(A).insertBefore(B) : A를 B의 요소 외 앞에 추가하는 법 - 친구

객체 복사와 제거 메소드

clone() : html 내부에 있는 요소를 복사는 메소드

remove() : html 내부에 있는 요소를 삭제하는 메소드 (이벤트, 데이터까지 모두 삭제)

detach() : html 내부에 있는 요소를 삭제하는 메소드 (이벤트, 데이터 삭제안함)

empty() : 내용만 삭제하는 메소드
