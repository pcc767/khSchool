2. 화면 배치요소 방법(block, inline)

- block : 한줄 또는 영역으로 지정하는 요소로 시작태그와 끝 태그가 항상 존재.
- inline : block 요소에 속해 있는 요소로 한 개의 요소를 수평으로 배치할때 사용.
- 실무 Tip : block요소 = div태그 / inline = span태그로 감싸서 활용.
- display : inline / block 요소의 속성을 변경할때 사용. inline-block / none(default) 속성이 중요하다.
