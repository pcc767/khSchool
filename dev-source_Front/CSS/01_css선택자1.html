css 선택자란?

- 특정한 html 태그를 선택하여 스타일 부여할때 사용되는 식별자(tag, #id, .class, 부모/후손, 이벤트등)


1. 모든 선택자

- html 문서안 모든 태그를 선택하는 기능


2. 태그 선택자

- 특정 태그를 모두 css로 적용하는 방법

    ul태그 전체(자식/손자 포함) 변경
  • li 영역

3. 아이디 선택자

- html 태그의 속성값 중 id를 지정하여 적용하는 방법

- id와 style 1:1 매핑되는 방법으로 많이 사용됨.

id1 영역입니다.
id2 영역입니다.

4. 클래스 선택자

- html 태그 속성중 class(그룹)으로 지정하여 스타일을 적용하는 방법

- class와 style이 1:n으로 매핑되는 방법으로 1:1매핑 가능. css 적용시 가장많이 활용됨.
