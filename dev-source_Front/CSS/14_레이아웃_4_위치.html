레이아웃 위치


1. 절대 / 상대 위치

- 절대위치 : 레이아웃 배치시 절대 위치(x,y)를 기반으로 태그를 배치시키는 방법

- 상대위치 : 부모 혹은 다른 레이아웃을 기준으로 상대적인 위치를 정하는 방법

절대위치1
절대위치2
상대위치1
상대위치2
- 디자인 계열 : margin, padding을 통해 위치 지정한다.
- 엔지니어(개발자) : 절대/상대 기준으로도 코딩하는 것이 익숙함.

2. 고정위치(fixed)

- 드래그가 가능한 긴 웹페이지에서 드래그가 되어도 특정 위치에 고정하는 속성

3. Visibility 속성

- 페이지에 특정 객체를 보이거나 보이지 않게 하는 속성
- 만일 보이지 않더라도 공간을 차기하는 옵션이 존재함.
- 또한 보이지 않는 경우, 버튼이면 이벤트가 발생하지 않은.

보이는 영역1

4. z-index

- 레이아웃간 보이는 우선순위를 설정하는 값(z값이라 불림)
- 값이 클수록 우선순위가 높아져 페이지 앞으로 보임

5. float 속성

- 페이지내의 요소간의 정렬 기준을 오른쪽/왼쪽으로 지정하는 방법
float영역1-left
float영역2-left
float영역3-right
float영역4-right
