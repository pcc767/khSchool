반응 선택자

- 사용자의 움직임에 따라 발생하는 이벤트에 반응하는 선택자.

- active : 사용자가 특정 태그를 클릭하였을때 발생

- hover : 사용자의 커서(마우스)가 특정 태그 위에 떠 있을때 발생

active-test 영역
hover-test 영역
